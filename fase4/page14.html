1. ¿Por qué Bootstrap es la opción preferida para los desarrolladores web?

Dividamos las ventajas en 8 partes:

1. Su cuadrícula receptiva


Ya no tendrá que pasar horas codificando su propia cuadrícula: Bootstrap viene con su propio sistema de cuadrícula predefinido. Ahora, puede comenzar directamente a llenar sus contenedores con contenido. Definir puntos de interrupción personalizados para cada columna es muy fácil utilizando sus interrupciones extra pequeñas, pequeñas, medianas, grandes y extra grandes. También puede simplemente ceñirse al valor predeterminado, ya que es posible que ya satisfaga las necesidades de su sitio.

2. Sus imágenes receptivas


Bootstrap viene con su propio código para cambiar el tamaño de las imágenes automáticamente en función del tamaño de pantalla actual. Simplemente agregue la clase .img-responsive a sus imágenes, y las reglas CSS predefinidas se encargan del resto. ¡Deje que Bootstrap cambie el tamaño de sus imágenes por usted! Incluso puede cambiar la forma de sus imágenes con la adición de clases como img-circle e img-round, y eso sin tener que ir y venir entre el código y su software de diseño.

3. Sus componentes


Bootstrap viene con una gran cantidad de componentes que puede agregar fácilmente a su página web, que incluyen: Barras de navegación Listas deplegables Barras de progreso Miniaturas No solo es muy sencillo agregar elementos de diseño llamativos a su página web, sino que también podrá estar seguro de que todos se verán geniales sin importar el tamaño de la pantalla o el dispositivo utilizado para verlos. Eso es una gran cantidad de funciones listas para usar al alcance de su mano. Para obtener una lista más completa de las funciones que se pueden agregar, consulte la documentación del componente.

4. Es JavaScript


¿Aún no tienes suficientes funciones? ¡Prueba JQuery !

Bootstrap también permite a los desarrolladores aprovechar más de una docena de complementos personalizados de JQuery. Esta biblioteca le brinda aún más espacio para jugar con la interactividad, ofreciendo soluciones fáciles para ventanas emergentes modales, transiciones, carruseles de imágenes y, uno de mis favoritos personales, un complemento llamado scrollspy , que actualiza automáticamente su barra de navegación a medida que se desplaza por una página. .

5. Su documentación

>En pocas palabras, la documentación de Bootstrap es una de las mejores que hemos visto. Cada fragmento de código se describe y explica en detalle explícito en su sitio web. Las explicaciones también incluyen ejemplos de código para la implementación básica, lo que simplifica el proceso incluso para los principiantes más principiantes. Todo lo que necesita hacer es elegir un componente, copiar y pegar el código en su página y modificarlo desde allí.

6. Su personalización


Una de las principales críticas cuando se trata de marcos como Bootstrap es su tamaño: el peso que arrojan puede ralentizar la aplicación en la primera carga. La versión actual del archivo CSS de Bootstrap, por ejemplo, es la friolera de 119 KB . Si bien esto puede no parecer especialmente grande en comparación con los archivos de imagen y video, para un archivo CSS, ¡eso es enorme! Sin embargo, lo que le permite hacer para combatir esto es personalizar qué funcionalidad desea incluir en su descarga. Con solo ir a su página Personalizar y descargar, puede marcar las funciones que no necesitará para su aplicación, recortando el peso de su archivo y ahorrando a sus usuarios el tiempo de carga adicional. ¡La personalización es clave!

7. Su comunidad


Al igual que con muchos proyectos de código abierto, Bootstrap tiene una gran comunidad de diseñadores y desarrolladores detrás. Estar alojado en GitHub facilita a los desarrolladores modificar y contribuir al código base de Bootstrap. También facilita que las personas colaboren, presten sus consejos e interactúen con compañeros y usuarios. Bootstrap tiene una página de Twitter activa, un blog de Bootstrap e incluso una sala dedicada a Slack . Y eso ni siquiera entra en la gran cantidad de desarrolladores dispuestos a ayudar con problemas técnicos en Stack Overflow, donde todas las preguntas se pueden encontrar bajo la etiqueta bootstrap-4 .

8. Sus plantillas externas


A medida que su popularidad creció, la gente comenzó a crear plantillas basadas en Bootstrap para acelerar aún más el proceso de desarrollo web. Hay muchos sitios web dedicados a compartir y comprar plantillas personalizadas basadas en Bootstrap.
